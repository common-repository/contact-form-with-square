== Description ==

Contact Form With Square is used to accept payment online via credit card.

It will works to accept simple payment from the customer without any e-commerce Website.

#### Free Features
* Simple Square payment gateway with Customer info.
* Sandbox support With the latest Version of SQUARE V2 API for testing in Version 2.5.5.
* User can Submit all the info and store into the database.
* Admin can see history of payments of their customer.
* Square card payment API is only available in US, Canada,UK, Australia and Japan.
* Notification will be sent to customer and admin.

### REQUIREMENTS:
* Get Square Account.
* Valid SSL certificate.
* WordPress 4.5+
* PHP version 5.6 or greater
